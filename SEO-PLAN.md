# SEO & Content Plan

Based on keyword research of what people in Algeria/MENA are searching for.

## High Priority

### 1. Homepage Copy — Target "create online store Algeria"
- Add keywords: "create online store", "no coding", "Arabic", "Algeria"
- Highlight COD analytics as a differentiator ("reduce failed deliveries", "track return rates")
- Mention upcoming payment methods (BaridiMob, CIB) to capture Algerian-specific searches

### 2. JSON-LD Structured Data on Homepage
- Add `SoftwareApplication` schema markup
- Helps Google show rich results (rating, pricing, category)
- Add to `app/layout.tsx` or `app/page.tsx`

### 3. French Landing Page (`/fr`)
- Algeria's primary web language is French, not English
- Duplicate homepage content in French
- Add `hreflang` alternates for SEO

## Medium Priority

### 4. Comparison Page (Leadivo vs Shopify vs Zid)
- High-intent keyword: people comparing platforms before choosing
- Target: "best ecommerce platform Algeria", "Shopify alternative Arabic"
- Route: `/compare` or `/vs-shopify`

### 5. Blog / Content Section
- Route: `/blog`
- First articles to write:
  - "How to start ecommerce in Algeria in 2026" (legal requirements, commercial register, CNRC)
  - "How to reduce COD return rates" (your analytics feature solves this)
  - "BaridiMob for ecommerce: what sellers need to know"
- These are exact phrases people are Googling with zero competition from platforms like yours

### 6. Docs SEO — Add Algerian-Specific Content
- Add a "How to start ecommerce in Algeria" guide to existing docs
- Cover: commercial register (CNRC code 607.074), legal requirements, payment methods
- Evergreen content that drives organic traffic

## Low Priority

### 7. Landing Page Variants by Country
- `/dz` for Algeria, `/ma` for Morocco, `/tn` for Tunisia, `/sa` for Saudi
- Each with localized copy, currency, and payment method references
- Helps rank for country-specific "ecommerce platform" searches

### 8. Customer Testimonials / Case Studies
- Social proof from real stores using the platform
- Target: "is Leadivo legit", "Leadivo reviews"

## Key Differentiators to Push in All Content
- COD analytics (competitors don't have this)
- Arabic-first / RTL native (not a plugin or afterthought)
- Mobile-first storefront (mobile is overtaking desktop in Algeria)
- Multi-language stores (Arabic + French + English)
- No coding required
